General Usage

Rollover Effect on buttons

  1. All buttons have the save rollover effect.

Function of Buttons

CANCEL

  1. Return to where user enters the page
  2. No action has been taken on any of the changed information

CREATE

  1. Direct user back to where he/she enters the page from
  2. After creation, on the enter page, a flash message will be displayed

SAVE

  1. Direct user back to where he/she enters the page from
  2. After saving, on the enter page, a flash message will be displayed

SAVE & CONTINUE

  1. Direct user to the next page to continue with the editing

ON/OFF

  1. Activate/Deactivate the media plan
