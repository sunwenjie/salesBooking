General Usage

Notice(Green)

  1. Use if it is successful updated or created

Info(Blue)

  1. Use if it is neither warning nor error message
  2. Just to state the findings

Warning(Yellow)

  1. Use for warning
  2. Highlight a message

Error(Red)

  1. Use for error finding

Conditions with close buttons

  1. Once click close, user will automatically agree not showing the message again.
  2. Suitable for messages that ask for further setup or setup updates
