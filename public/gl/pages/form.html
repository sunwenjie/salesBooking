General Usage

Mandatroy fields

  1. All mandatory fields should be indicated with the “ * ”. For example: Name*

Field lengh

  1. Length of input fields should communicate the intended content.
  2. • Fields don’t need to all be the same length. Mix long and short field length according to the content it is used for • You can override input field widths if you have specific character length requirements for the field (eg. IP)

Notice(Green)

  1. Use if it is successful updated or created

Text alignment

  1. “Budget field” or “% or currency field” – Right align “Slider” or “Bulk upload” – Center align
  2. All other fields – Left align

Add (Single Page)

General Usage

Buttons

CANCEL

  1. Return to where user enters the page
  2. No action has been taken on any of the changed information

CREATE

  1. Direct user back to where he/she enters the page from
  2. On the enter page, a flash message will be displayed either the creation was successful or not.

General Usage

Buttons

CANCEL

  1. Return to where user enters the page
  2. No action has been taken on any of the changed information

CREATE

  1. Direct user back to where he/she enters the page from
  2. On the enter page, a flash message will be displayed either the creation was successful or not.

Add (MULTI-PAGE, FORM CAN BE INCOMPLETE WHEN CREATING, WITH ON/OFF BUTTON)

General Usage

Buttons

CANCEL

  1. Present on all pages of the form
  2. Return to where user enters the page
  3. No action has been taken on any of the changed information

CREATE

  1. Present only on the last page of the form
  2. Direct user back to where he/she enters the page from
  3. On the enter page, a flash message will be displayed in either one of the following options:
    • a) Media plan was successfully created. however the setup is incomplete, please complete the setup for approval.
    • b) Media plan was successfully created and is waiting for activation.

ON/OFF

  1. Maintain to be OFF
